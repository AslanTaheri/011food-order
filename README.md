# AvocaDoor 🥑

AvocaDoor is a headless ecommerce solution that serves as a grocery shopping storefront. 🏪 It can be connected to Shopify or your prefered backend provider to serve your business logic needs. 🛒

# Tech Stack

- React
- CSS
- Deployed on Netlify
